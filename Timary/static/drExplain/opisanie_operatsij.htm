4 ОПИСАНИЕ ОПЕРАЦИЙ
4.1 Регистрация пользователя
Условия выполнения :
  • • 
    Доступ к интернету.
  • • 
    Валидные данные (email, пароль, роль).
     
Подготовительные действия :
1. Открыть страницу регистрации.
2. Выбрать вход или остаться на странице регистрации.
 
Основные действия :
1. Ввести email и пароль.
2. Ввести имя и логин.
3. Нажать кнопку «Зарегистрироваться».
 
 
 
Заключительные действия :
  • • 
    Переход на главную страницу после активации аккаунта.
 
Ресурсы :
  • • 
    База данных (запись нового пользователя).
 
4.2 Создание расписания (для учителя)
Условия выполнения :
  • • 
    Пользователь с ролью «Учитель».
  • • 
    Авторизация к группе, для которой создается расписание.
 
Подготовительные действия :
1. Авторизоваться в системе.
2. Перейти в раздел «Расписание».
 
Основные действия :
1. Нажать на "+"
2. Указать нужную неделю, день недели, время, предмет и кабинет.
3. Нажать «Сохранить».
 
 
Ресурсы :
  • • 
    Серверная часть (API для обновления расписания).
  • • 
    База данных (таблица с расписанием).
     
     
4.3 Добавление домашнего задания
Условия выполнения :
  • • 
    Пользователь с ролью «Учитель».
  • • 
    Выбранная группа и предмет.
 
Подготовительные действия :
1. Перейти в раздел «Домашние задания».
2. Нажать «+».
 
Основные действия :
1. Ввести описание, срок сдачи и указать готовность
2. Выбрать предмет.
3. Нажать «Добавить».
 
 
Ресурсы :
  • • 
    База данных (таблица с д.з.).
  • • 
    Серверная часть (API для сохранения задания).
